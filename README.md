# Loan Calculator Widget

This repository contains a standalone, embeddable loan calculator widget built with plain HTML, CSS, and JavaScript.

## Run locally

1. Clone the repository.
2. Open `index.html` in your browser.

No build step or dependencies are required.

## Deploy with GitHub Pages

1. Push this repository to GitHub.
2. In the repository settings, open **Pages**.
3. Set the source to deploy from the main branch root (`/`).
4. Save and wait for GitHub Pages to publish.

The widget page will be available at your GitHub Pages URL.

## Embed in another webpage

### Option 1: iframe embed

```html
<iframe
  src="https://<your-github-pages-url>/"
  title="Loan Calculator"
  style="width: 100%; min-height: 900px; border: 0;"
></iframe>
```

### Option 2: copy files directly

Copy these files into your website project and include them:

- `index.html`
- `styles.css`
- `loan-calculator.js`
